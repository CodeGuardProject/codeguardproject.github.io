KU Leuven — DistriNet, iDeas Research Group

The iDeas research group, led by Prof. Dr. Bert Lagaisse, specializes in designing intelligent, secure, and distributed software solutions. The group combines fundamental research with practice-oriented applications and has a strong focus on supporting software developers in building secure, reliable, and efficient software, both manually and with the help of AI code assistants.

Industry

Advisory Board

The project is guided by an advisory board of industry partners who provide practical insights, real-world use cases, and validation of research outcomes. The board is composed of companies that want to enable secure code generation when integrating large language models (LLMs) into their software development lifecycle, including:

  • AI project developers — benefit from early detection of vulnerabilities in AI-generated code
  • Software engineers — adopt state-of-the-art guardrails for LLM-assisted development in client engagements
  • AI engineers — ensure that LLM-integrated pipelines produce reliable, secure, and compliant code
